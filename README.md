# umbracoTV

# Live Site 

https://umbratv-837098036726.us-central1.run.app/

A simple IPTV streaming platform built with Umbraco CMS.

## Features
- TV channel management with logo support (media picker or URL)
- M3U playlist integration
- Channel categories
- EPG (Electronic Program Guide) support
- Responsive video player

## Tech Stack
- Umbraco CMS
- .NET 10.0
- SQLite database

## Getting Started
1. Clone the repository
2. Navigate to `umbraTV` folder
3. Run `dotnet restore`
4. Run `dotnet run`
5. Access at `https://localhost:44337`
